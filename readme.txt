== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `plugin folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Your Admin Theme is ready
4. For Better use also download http://wordpress.org/plugins/user-role-editor/ to set Shop Manager Account.

== Frequently Asked Questions ==

= What is the difference between Shop Manager and admin =

Shop Manger can add, edit, modify and delete Orders, Customers, Products but he will not be allowed to affect core functionalities of wordpress like modification of wordpress theme files.

= Why User Role Editor Plugin =

Use this plugin to set preveliges for Shop Manager


= What about Design =

A very beautiful design is ready to attract Shop Manager.
